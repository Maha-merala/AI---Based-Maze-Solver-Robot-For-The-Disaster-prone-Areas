# AI-Based Maze Solver Robot For The Disaster Prone Areas
Disaster-prone environments such as collapsed buildings, flood zones, and fire-affected structures pose extreme risks to human rescue teams due to unstable terrain and limited visibility. This project presents an AI-based maze solver robot designed to navigate complex, unknown environments representative of such disaster scenarios. The proposed system integrates computer vision techniques with classical pathfinding algorithms to analyse a physical maze of 120 cm × 130 cm, divided into a grid of cells ranging from 30 to 35 cm each. A maze image is captured and processed using Gaussian thresholding and skeletonization to extract navigable paths, after which Breadth-First Search (BFS) is applied to determine the optimal route from start to goal.The computed path is translated into a compact instruction sequence  comprising forward, backward, and turn commands and transmitted to an ESP32 microcontroller, which drives a dual-motor robot via an L298N motor driver. Ultrasonic sensors mounted on all four sides provide real-time obstacle detection to prevent collisions during execution. The system automatically generates Arduino-compatible code with calibrated step timing based on physical cell dimensions and robot speed. 
	Experimental results demonstrate accurate path following with a combined accuracy metric incorporating path efficiency, skeleton coverage, and motion smoothness. This solution offers a scalable, low-cost framework for autonomous robot navigation in GPS-denied, hazardous disaster environments, with potential integration into search-and-rescue operations.
